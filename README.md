# Cookies Order App 🍪

## Deskripsi Aplikasi

Cookies Order App adalah aplikasi berbasis mobile/web yang dibuat menggunakan **Flutter** dan terhubung dengan **Supabase** sebagai database backend.
Aplikasi ini digunakan untuk membantu mengelola pesanan cookies secara sederhana, mulai dari menambahkan pesanan, melihat daftar pesanan, melihat detail pesanan, hingga menghapus pesanan.

Dengan aplikasi ini, pengguna dapat mencatat pesanan pelanggan dengan lebih terstruktur sehingga mempermudah pengelolaan data pesanan cookies.

---

## Fitur Aplikasi

### 1. Menambahkan Pesanan

Pengguna dapat menambahkan pesanan baru dengan mengisi:

* Nama pelanggan
* Tanggal pengambilan
* Jenis cookies
* Jumlah pesanan
* Harga cookies

Data yang dimasukkan akan langsung tersimpan ke database Supabase.

### 2. Menampilkan Daftar Pesanan

Aplikasi menampilkan seluruh data pesanan dalam bentuk list sehingga pengguna dapat melihat semua pesanan yang sudah tersimpan.

### 3. Melihat Detail Pesanan

Pengguna dapat melihat detail pesanan yang berisi:

* Nama pelanggan
* Tanggal pengambilan
* Jenis cookies
* Jumlah pesanan
* Harga
* Total harga pesanan

### 4. Mengedit Pesanan

Pengguna dapat memperbarui data pesanan yang sudah tersimpan.

### 5. Menghapus Pesanan

Pengguna dapat menghapus pesanan yang tidak diperlukan dari database.

## Widget yang Digunakan

Beberapa widget Flutter yang digunakan dalam aplikasi ini antara lain:

* **MaterialApp**
  Digunakan sebagai root aplikasi Flutter yang menggunakan Material Design.

* **Scaffold**
  Digunakan untuk membuat struktur dasar halaman seperti AppBar, body, dan FloatingActionButton.

* **AppBar**
  Digunakan untuk menampilkan judul halaman aplikasi.

* **Text**
  Digunakan untuk menampilkan teks seperti nama pelanggan dan informasi pesanan.

* **TextFormField**
  Digunakan sebagai input form untuk mengisi data pesanan.

* **Form & GlobalKey**
  Digunakan untuk validasi form sebelum data disimpan.

* **ElevatedButton**
  Digunakan sebagai tombol untuk menyimpan data dan menambah item.

* **FloatingActionButton**
  Digunakan untuk menambahkan pesanan baru.

* **ListView.builder**
  Digunakan untuk menampilkan daftar pesanan secara dinamis.

* **Card**
  Digunakan untuk menampilkan data pesanan dengan tampilan yang lebih rapi.

* **ListTile**
  Digunakan untuk menampilkan informasi pesanan dalam list.

* **Icon & CircleAvatar**
  Digunakan untuk menampilkan ikon pada list pesanan.

* **Navigator**
  Digunakan untuk berpindah halaman (Home, Form, Detail).
